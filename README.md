## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Open the project folder:
   ```bash
   cd Playground
   ```
3. Start a local server (Python example):
   ```bash
   python -m http.server 8000
   ```
   Then navigate to `http://localhost:8000` in your browser.

## Usage

- Edit `index.html`, `style.css`, or `script.js` and refresh the page to see changes instantly.
- The live preview updates automatically if you integrate a live‑reload tool.
